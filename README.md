zxsq_goxiti
===========

discuz插件，在线围棋习题
本插件将 "网弈习题" 集成到discuz，定义了一个新的discuz标签 [goxiti][/goxiti]，只需将习题的SGF代码粘贴到标签中即可，大大方便了棋谱交流。

使用方法
1. 配置插件，设置允许使用的论坛版块

2. 用记事本打开SGF棋谱

3. 全选SGF棋谱文本内容，粘贴到编辑器棋谱发布工具栏中
